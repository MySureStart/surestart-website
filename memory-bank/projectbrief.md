# Project Brief: SureStart Website

## Overview

**Project:** Website migration and modernization for SureStart  
**Type:** Static HTML/CSS/JS website  
**Status:** Active migration from Squarespace

## Domain Information

| Domain | Purpose | Status |
|--------|---------|--------|
| `mysurestart.com` | Legacy Squarespace site | Being retired |
| `mysurestart.org` | New static HTML site | Active, primary domain |

## Mission

SureStart is an AI education company that partners with K-12 schools and higher education institutions to deliver small-group, mentor-led AI programs. The website serves as the primary marketing and information hub for students, educators, and institutional partners.

## Migration Goals

1. **Modernize Tech Stack** - Move from Squarespace CMS to static HTML for better performance and control
2. **Improve SEO** - Add missing meta tags, canonical URLs, and structured data
3. **Preserve Link Equity** - Implement 301 redirects from all legacy URLs
4. **Enhance User Experience** - Streamlined navigation and clearer calls-to-action
5. **Maintain Brand Consistency** - Consistent design language across all pages

## Key Stakeholders

- **SureStart Leadership** - Content approval, strategic direction
- **Development Team** - Implementation, deployment
- **Marketing Team** - Content, SEO, analytics
